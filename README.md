# GAME_PROGRAM-EX--1
## EXP:1 Implementing various effects in a material such as emissive, roughness and metallic properties in Unreal Engine.
### NAME: VIJAYAKUMAR S
### REG NO: 212224040359

# Procedure
Open Unreal Engine and create a new material.
Open the Material Editor from the Content Browser.
Add a color node and connect it to Base Color.
Add nodes for Emissive Color to create a glowing effect.
Add a scalar value to the Roughness input to control surface smoothness.
Add a scalar value to the Metallic input to create a metallic appearance.
Apply and save the material.
Drag and apply the material to a 3D object in the scene.
